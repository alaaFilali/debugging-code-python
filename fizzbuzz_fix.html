اصلاح اخطاء تمرين FizzBuzz

التعليمات:

  • اقرأ الكود جيدا

  • ابحث عن الخطأ

  • عدل على الكود واصلح الخطأ

  • ممنوع نسخ ولصق حل من جهة اخرى

هذا البرنامج سوف يقوم بطباعة كل الاعداد من 1 الى 100 بالترتيب فاذا كان العدد يقبل القسمة على 3 اذن سوف يطبع الجملة Fizz بدل العدد واذا كان العدد يقبل القسمة على 5 سوف يقوم بطباعة الجملة Buzz اما اذا كان يقبل القسمة على 3 و 5 معا اذن سوف يقوم بطباعة الجملة FizzBuzz 


for number in range(1, 101):
    if number % 3 == 0 or number % 5 == 0:
        print("FizzBuzz")
    if number % 3 == 0:
        print("Fizz")
    if number % 5 == 0:
        print("Buzz")
    else:
        print(number)

الحل:

هناك اكثر من خطأ في الكود وسوف نصلحهم واحد بواحد, فلو نشغل البرنامج سوف نلاحظ انه لا يعمل كما نريد نحن مثلا في مكان العدد 3 هناك الجملة FizzBuzz بدل Fizz ونفس الشيئ مع العدد 5 قام بطباعة الجملة FizzBuzz بدل Buzz

الجملة FizzBuzz لا يجب ان تظهر الا اذا كان العدد يقبل القسمة على 3 و 5 معا

باستعمال عبارة print سوف تساعدنا كثيرا وسوف تظهر لنا الخطأ 


for number in range(1, 101):
    print(number)
    if number % 3 == 0 or number % 5 == 0:
        print("FizzBuzz")
    if number % 3 == 0:
        print("Fizz")
    if number % 5 == 0:
        print("Buzz")
    else:
        print(number)

بعد تشغيل البرنامج نلاحظ ان للعدد الواحد يتم طباعة اكثر من عبارة, سوف نبدأ بالمشكل الاول وهو عبارة FizzBuzz المشكل هو انه يجب ان يكون العدد يقبل القسمة على 3 و 5 وليس أو 5 اذن في مكان or نضع and والمشكل الثاني هو ان عبارات ال if منفصلة عن بعضها البعض اي يمكن للعدد ان يقبل القسمة على 3 و 5 و 15 في نفس الوقت وسوف يتم طباعة جميع الجمل اضافة الى ان عبارة else تنفذ فقط في حالة عدم تحقق شرط ال if الاخيرة وهذا مالانريده نحن, نحن نريد ان يتم التحقق من الشرط الاول ان لم يتحقق في تلك الحالة ننتقل الى الشرط الثاني والثالث ثم ان لم يتحقق اي شرط نذهب الى عبارة else اذن نحولها جميعا الى elif 


for number in range(1, 101):
    if number % 3 == 0 and number % 5 == 0:
        print("FizzBuzz")
    elif number % 3 == 0:
        print("Fizz")
    elif number % 5 == 0:
        print("Buzz")
    else:
        print(number)
